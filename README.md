# 灌顶与功课记录 App（PWA 本地版）

这是一个手机优先的 PWA，本地保存数据，不需要 App Store、不需要服务器、不需要登录。当前版本默认空白数据，不再预置示例/测试记录。

## 已实现模块

- **日历**：按日期查看灌顶 / 口传 / 开示 / 法会 / 共修，也可以记录未来活动。
- **记录**：搜索和筛选受法记录，支持富文本「灌顶观想内容」。
- **功课**：按心咒 / 修法分组计数，支持加行这类父子功课，并支持每日/每周周期目标。
- **法本 / 链接**：保存官网、PDF、视频、报名链接、Google Drive、备注等链接资料。
- **设置**：JSON 完整备份 / 恢复、CSV 导入导出、类型颜色自定义、默认设置。

## 数据保存

数据保存在当前浏览器的 `localStorage`。更换手机、清理浏览器缓存、更换网址前，请先在「设置」里导出完整 JSON 备份。

## iPhone 添加到主屏幕

部署到 GitHub Pages / Netlify / Vercel 后，用 Safari 打开网址，点击分享按钮，然后选择「添加到主屏幕」。

## 本地预览

直接打开 `index.html` 即可预览。部分 PWA 离线缓存功能需要通过网页服务器访问才会启用。


## Icon cache fix
This build includes root-level `apple-touch-icon.png`, `om-apple-touch-icon.png`, `favicon-192.png`, and `icon.png` for iOS home screen icon detection.
